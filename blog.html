Var has the function or global scope. It gets hoisted to the top of its scope and initialized undefined. It can be updated or re-declared.Let have the block scope. It also got hoisted to the top of its scope but didn't initialize. It can only be updated and can't be re-declared.Const variable has the block scope. It also got hoisted to the top of its scope but didn't initialize. It can't be updated or re-declared.

The arrow function allows a developer to accomplish the same result with fewer lines of code and approximately half the typing than regular function. Regular functions created using function declarations or expressions are constructible and callable. Since regular functions are constructible, they can be called using the 'new' keyword. Regular function can have duplicate names but arrow functions can never have duplicate named parameters, whether in strict or non-strict mode.

Template strings allows us to create strings that are complex and contain dynamic elements. With template strings, we can add something called tags in front of a template string. They are functions that take a string and the decomposed parts of the string as parameters. Another great feature of template strings is that we can have strings with multiple lines for better code readability.
